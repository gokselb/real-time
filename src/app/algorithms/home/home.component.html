This app is built to help your data set tested on RM and EDF.

Functionalities

  • RM and EDF
  • Multiple tasks
  • Add task
  • Remove task
  • Set custom deadline for each task
  • See the result as scheduled tasks and utilization

Simply, select the algorithm that you desired to test and enter your data set. If you need to remove a task from data set use 'X' button next to task. You can add more tasks using 'Add Task' button. Click on 'Schedule' button to see the result. Result will be shown as scheduled tasks along with utilization value. You can not see the scheduled result if utilization is not satisfied. If utilization is not satisfied app can not schedule this data set.
